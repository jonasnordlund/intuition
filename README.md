# intuition

Simple single page application with interactive drawing tools, etc. to generate JSON suitable for image generation with Ideogram v4.

This JSON can later be used in e.g. a CLI tool that calls the Generate API endpoint.

I first intended to take this the whole nine yards with API calling, but thought it's be easier for redistribution and _ad hoc_ use to foucs on the JSON (the hard part), leaving the user free to combine this tool with anything else.
